Typography

One ramp, one set of semantic roles shared across app, website, blog and docs. Sizes are rem with paired line-heights; roles set size, leading, weight and tracking only — the surface's font-family (Inter on web, system in app) is inherited. Display tier and lead are fluid via clamp(). Resize the window to watch them scale.
